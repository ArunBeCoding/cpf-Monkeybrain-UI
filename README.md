## cpf-MonkeyBrainUI
A webapp optimizing cpf's current website for ease of use, as well as to provide analytics and forecasting for citizens' finances

### Initialization
`git clone https://github.com/Ivanlxw/cpf-Monkeybrain-UI.git`

## Front end
`cd front-end`

`npm install`

`npm start`

## Back end

`cd back-end`

`node index.js`
